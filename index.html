Данное руководство описывает мой опыт сборки аналога подсветки Ambilight которая используется на телевизорах Phillips. Устройство состоит из светодиодной ленты которая закреплена по всему периметру телевизора с тыльной его стороны. Лента подключается к одноплатному компьютеру Raspberry Pi через микроконтроллер Esp32 с драйвером HyperSPI. При помощи программы HyperHDR анализируется видео и аудио поток входного сигнала полученный с Usb-карты захвата. После анализа параметры цвета передаются на светодиодную ленту которая воспроизводит рассеянный свет по периметру ТВ благодаря чему поверхность стены за его корпусом динамически освещается, тем самым дополняя ореолом изображение на экране и как бы визуально увеличивая его размер.

Процесс сборки и настройки разбит на следующие этапы:

  • Покупка комплектующих:
    • Устройство для хостинга HyperHDR
    • Светодиодный драйвер
    • HDMI сплиттер
    • Usb карта захвата
  • Подготовка платы светодиодного драйвера
  • Установка Raspberry Pi OS с HyperHDR
  • Прошивка платы светодиодного драйвера
  • Установка светодиодной ленты на ТВ
  • Общая сборка компонентов
