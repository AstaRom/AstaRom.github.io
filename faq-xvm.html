Как активировать отображение статистики в моде XVM?

 

 

Пройти процедуру активации сетевых сервисов на сайте комплексной модификации XVM по следующей инструкции.

Как изменить отображаемый рейтинг эффективности?

 

 

Рейтинг эффективности выбирается в личном кабинете на официальном сайте мода www.modxvm.com.

Как включить/отключить вывод дополнительных элементов в конфиге?

 

 

Боевой интерфейс:
 
* "battleLabels" / hit_log - панель лога нанесенного урона
* "battleLabels" / total_efficiency - панель общей эффективности игрока в бою
* "battleLabels" / damage_log - дамаг панель
* "battleLabels" / repair_timer - таймер ремонта модулей
* "battleLabels" / info_panel - инфо панель
* "battleLabels" / total_hp_panel - панель общего запаса прочности техники (ХП команд)
* "battleLabels" / color_blind - вкл/выкл режима цветовая слепота
 
Маркеры техники:
 
* "markers" / squad_number - номер взвода в маркерах
* "markers" / lowHp_marker - маркер приоритета цели (с менее 25% прочности)
 
Панели команд:
 
* "playersPanel" / xmqp_marker - маркер сервиса XMQP
* "playersPanel" / spotted_marker - маркер засвеченных противников
* "playersPanel" / hp_panels - запас прочности техники (ХП техники), отдельно для каждого режима панелей команд

 

Открыть файл по пути:

 

{{клиент игры}} / res_mods / configs / xvm / NDO / @settings.xc

 

Прописать значение true для включения элемента, false - для отключения.

Как изменить время отображения иконки перка "Шестое чувство"?

 

 

Открыть файл по пути:

 

{{клиент игры}} / res_mods / configs / xvm / NDO / battle.xc

 

Отредактировать выделенные фрагменты кода:

 

"battle": {
 "sixthSenseDuration": 2000,
},

 

Меняем значение 2000 (2 сек), 10000 (10 сек) и т.д.

Как поменять клавишу увеличения миникарты с "Caps Lock" на другую?

 

 

Открыть файл по пути:

 

{{клиент игры}} / res_mods / configs / xvm / NDO / hotkeys.xc

 

Отредактировать выделенные фрагменты кода:

 

"minimapZoom": {
 "enabled": true, "keyCode": 58, "onHold": true }
},

 

Вместо значения 58 выбираем код нужной вам клавиши (код см. в данной таблице).

Параметр "onHold" отвечает за "залипание" клавиши: меняем true на false для отключения данной функции.

Параметр "enabled" отвечает за возможность увеличения миникарты: меняем true на false для отключения данной функции.

Как изменить количество рядов в карусели техники в ангаре?

 

 

Открыть файл по пути:

 

{{клиент игры}} / res_mods / configs / xvm / NDO / carousel.xc

 

Отредактировать выделенные фрагменты кода:

 

"carousel": {
 "rows": 0,
},

 

Меняем значение 0 задавая необходимое вам количество рядов.

При нулевом значении количество рядов будет регулироваться в настройках клиента.

Как изменить положение часов в ангаре?

 

 

Открыть файл по пути:

 

{{клиент игры}} / res_mods / configs / xvm / NDO / clock.xc

 

Отредактировать выделенные фрагменты кода:

 

"clock": {
 "x": -300,
 "y": 0,
 "align": "center",
},

 

Меняем значение -300 задавая необходимые координаты по оси X.

Меняем значение 0 задавая необходимые координаты по оси Y.

Параметр center задавая выравнивание часов в окне клиента, может иметь следующие значения:

 

  right - выравнивание элементов по правому краю экрана;

  left - выравнивание элементов по левому краю экрана;

  center - выравнивание элементов по центру экрана.

Выбрал при установке мод "Инфо панель", но в бою он не работает, почему?

 

 

Модификация включается по горячей клавише (по умолчанию это кнопка «Alt»). Возможны 2 состояния:

 

  1) Прицел наведен на контур союзника/противника - отображаются характеристики техники в прицеле;

  2) Прицел не наведен - отображаются характеристики вашей техники.

 

Задаем свою горячую клавишу, открыть файл по пути:

 

{{клиент игры}} / res_mods / configs / xvm / NDO / battleLabelsTemplates.xc

 

Отредактировать выделенные фрагменты кода:

 

"info_panel_bg": {
 "hotKeyCode": 56,
 "onHold": true,
},

"info_panel": {
 "hotKeyCode": 56,
 "onHold": true,
},

 

Вместо значения 56 выбираем код нужной вам клавиши (код см. в данной таблице).

Параметр "onHold" отвечает за "залипание" клавиши: меняем true на false для отключения данной функции.
